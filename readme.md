# 使用须知

## 基于爬虫+GUI开发

### 第一步: pip install -r ./requirements.txt **下载**文件中所需的第三方模块(没有配置Python解释器和环境的可以自己百度配一下~)

### 第二步: 运行spider中的crawl.py文件将英雄的皮肤爬取进本地，当然也可以部署到数据库当中，同学们可以自行选择

### 第三步: 运行run.py文件即可！
